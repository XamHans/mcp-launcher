# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-12

### Added
- Initial release of MCP Launcher
- Web dashboard for managing deployments
- Real-time deployment tracking via WebSockets
- One-click Cloud Run deployments
- Built-in MCP server template
- CLI tool (`mcp-launcher` and `deploy-mcp` commands)
- GCP logging and metrics integration
- Prerequisites checking in CLI
- Environment variable validation

### Features
- Express + Socket.io backend
- React/Vite frontend with Preact
- Tailwind CSS styling
- Automatic browser opening
- Support for custom MCP servers
- Artifact Registry integration
- Cloud Run deployment automation
